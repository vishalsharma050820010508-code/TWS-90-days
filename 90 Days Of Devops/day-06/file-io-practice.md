# Day 06 – Linux Fundamentals: Read and Write Text Files

## Task

- Creating a file
- Writing text to a file
- Appending new lines
- Reading the file back

## Commands Used

- `touch notes.txt`
Glimpse: To text file 


- `echo "Hi, I am Danish Sayyed" > notes.txt`
Glimpse: Add line in file 

- `echo "I will be Devops 2026" >> notes.txt`
Glimpse:  Append line in notes.txt
- `echo "My Dream is to work in Temporal, I commit myself will one day" >> notes.txt `
Glimpse:  Append line in notes.txt

- `cat notes.txt | head -n 1`
Glimpse: To read notes.txt heading  line 1

- `cat notes.txt | tail -n 2`
Glimpse: To read notes.txt last 2 lines 

- `echo "Hands-on is key for Devops" | tee -a notes.txt`
Glimpse: Append and prints the output to terminal

- ` cat notes.txt`
Glimpse: Read notes.txt
